Real View already reads embedded metadata and checks image source domains against a huge blocklist, on every page, for free. But a lot of AI art — especially on Pinterest — is uploaded directly with no external source link at all, and no metadata left in the file. There's nothing for those checks to go on. The visual model looks at the actual pixels instead, so it doesn't need either one.

It's opt-in specifically because it's a real ~320MB download the first time and uses more CPU per image afterward. Once it's downloaded, it runs entirely in your browser — nothing is ever uploaded, and it keeps working offline.

Change this anytime from the extension's Options page. You can also turn Real View off entirely for specific sites from the toolbar popup.
