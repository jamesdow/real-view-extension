HOW IT WORKS

Three layers of signal, side by side.

No single trick catches everything — so Real View runs a stack of checks, lightest first, and only reaches for the heavier one when it has to.

Real View: checking 24 images…

You'll see this in the corner of the page while it works. Most pages resolve in a second or two — a big, image-heavy page just takes a beat longer. Nothing's frozen, it's working through what's actually on screen.

01 / METADATA & SOURCE

Metadata & source signals

Reads embedded generator tags left in the file itself, and cross-references the image's source domain against a huge, refreshable list of known AI platforms. Near-instant, and costs nothing.

no setup · always on
✓
02 / VISUAL MODEL

On-device visual model

When metadata and domain give no answer, an ML model examines the actual pixels — right there in your browser. Nothing is ever uploaded or sent anywhere. In our own testing it calls roughly 84% of images correctly — not perfect, and it stays quiet rather than guess when unsure.

opt-in · private

Right-click any image to flag a miss or fix a false positive. Your corrections are remembered instantly, for you — no waiting on anyone else.

QUESTIONS

Before you install.

Will this slow down my browsing?
No. Metadata and domain checks are near-instant. The optional visual model only ever runs on images actually visible on your screen, a couple at a time — never a whole page's backlog at once.
Does it work on sites other than Google, Bing, DuckDuckGo and Pinterest?
Yes — those four get dedicated source-resolution because they hide an image's real origin behind a proxy. Every other site still gets metadata and domain-list detection out of the box.
What happens to images it's not sure about?
It leaves them alone. An inconclusive image is shown exactly as normal — Real View never guesses.
Is my data ever sent anywhere?
No. The visual model runs entirely on-device via WebAssembly. There's no server component to send anything to, even if we wanted to.
How do you make money from this?
I don't, really — Real View is an independent, one-person project, not a business. If it saves you the scroll, you're welcome to buy me a coffee or back it on GitHub Sponsors. No account, no catch, no upsell hiding behind it.
